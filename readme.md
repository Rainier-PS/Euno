# StudyHub

StudyHub is a comprehensive student productivity application designed to help students manage their academic and personal well being through integrated tools for tracking habits, journaling, studying, and wellness activities.

## Demo

Visit the live demo at https://rainier-ps.github.io/Study-Hub/

## Features

### Home Dashboard
Personalized greeting with time based welcome message, mood preview, quick statistics, weekly mood visualization, insights teaser, and coin balance display.

### Daily Check In
Track your daily mood with emoji selection and slider, energy level, sleep hours, stress level, and thoughts entry. The system calculates a wellness score and tracks your streak.

### Journal System
Notes with markdown editor, labels, reminders, search and sort functionality, and draft auto save. Journal entries with the same features plus dedicated journaling interface. To Do list with priorities, deadlines, status tracking, search and sort, and automatic cleanup. Habit tracker with weekly grid, completion tracking, metrics, and CSV export.

### Insights
Summary statistics, mood charts for 7 day and 30 day periods, distribution analysis, habit consistency charts, wellness streaks, emotional patterns, weekly averages, and PDF report generation.

### Wellness
Breathing exercises with 4-7-8, Box, and Calm techniques with visual feedback. Gratitude journal with 3 daily entries and history. Self care challenges with daily micro challenges and completion tracking. Reflection prompts with rotating prompts for journaling.

### Study Tools
Pomodoro timer with focus, short break, and long break modes plus session tracking. Stopwatch with lap tracking and display updates. Countdown timer with custom duration and browser notifications. Calendar with month, week, day, and year views, event management, and notifications. Flashcards with deck management, card review, and correct or wrong tracking. Music player with Lo Fi YouTube embeds and track switching.

### Shop
StudyCoins system with item purchasing, inventory management, and coin history. Power ups include streak freeze, double coins, lucky spin, and focus boost.

### Settings
Profile management with name and gender. Theme selection with light, dark, and system options. Data export and import. Data clearing. Onboarding restart.

### Onboarding
Interactive tour with spotlight highlighting and step navigation to help new users get started.

### Navigation
Sidebar navigation for desktop, mobile drawer, bottom navigation bar, and page routing.

## Technology

StudyHub is built with vanilla JavaScript using ES modules, modular CSS, and localStorage for data persistence. The application uses jsPDF for report generation and Material Icons for the icon system.
